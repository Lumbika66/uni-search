# uni-search
🔍 Universal File Searcher
Универсальная программа для поиска текста в файлах и архивах с графическим интерфейсом и поддержкой множества форматов.

https://img.shields.io/badge/Python-3.7+-blue.svg
https://img.shields.io/badge/License-MIT-green.svg
https://img.shields.io/badge/GUI-Tkinter-orange.svg

✨ Возможности
📁 Поддерживаемые форматы файлов
Текстовые файлы: .txt, .csv, .json, .xml, .html, .py, .js, .md, .log, .ini, .cfg

Архивы: .zip, .rar, .tar, .gz (поиск внутри без распаковки!)

🔎 Функции поиска
📝 Обычный текстовый поиск - быстрый поиск по ключевым словам

🧮 Регулярные выражения - мощный поиск по шаблонам

🔠 Регистрозависимый поиск - опция учета регистра

📂 Рекурсивный поиск - поиск во вложенных папках

🎯 Поиск по шаблону - фильтрация по расширению файлов (*.txt, *.py и т.д.)

🗂️ Поиск в архивах - автоматический поиск в ZIP, RAR, TAR архивах

🖥️ Интерфейс
Графический интерфейс (GUI) - удобный интерфейс на Tkinter

Консольная версия - для любителей командной строки

Цветовое кодирование - архивы выделяются цветом

Вкладки результатов - удобная навигация между результатами

Экспорт результатов - сохранение результатов в файл

📦 Установка
Требования
Python 3.7 или выше

Установленные архиваторы для соответствующих форматов

Установка зависимостей
bash
pip install rarfile
Для работы с RAR файлами (дополнительно):
Windows: Установите WinRAR или 7-Zip

Linux: sudo apt-get install unrar или sudo apt-get install p7zip-full

Mac: brew install unrar

📚 Примеры использования
1. Поиск email адресов в директории
python
# Регулярное выражение для email
pattern = r'[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}'
searcher.search_in_directory("/папка/с/файлами", pattern, regex=True)
2. Поиск телефонных номеров в архиве
python
# Российские номера телефонов
pattern = r'\+7\s?\(?\d{3}\)?\s?\d{3}[\s-]?\d{2}[\s-]?\d{2}'
searcher.search_in_file("архив.zip", pattern, regex=True)
3. Поиск конкретного слова в текстовых файлах
python
# Поиск слова "договор" с учетом регистра
searcher.search_in_directory("/документы", "договор", 
                           case_sensitive=True, 
                           file_pattern="*.txt")

# Поиск в одном файле

# Поиск в директории
search_in_directory(directory, pattern, file_pattern="*", 
                   recursive=True, case_sensitive=False, regex=False)

# Поиск в тексте
search_in_text(text, pattern, case_sensitive=False, regex=False)

🐛 Устранение неполадок
Проблема: "ModuleNotFoundError: No module named 'rarfile'"
Решение:

bash
pip install rarfile
Проблема: RAR файлы не читаются
Решение: Установите unrar/winrar:

Windows: WinRAR

Linux: sudo apt install unrar

Mac: brew install unrar

Проблема: Кириллица отображается неправильно
Решение: Программа автоматически пробует кодировки UTF-8, CP1251, latin-1

Проблема: Большие файлы обрабатываются медленно
Решение: Используйте более конкретные шаблоны поиска или фильтруйте по размеру файлов
